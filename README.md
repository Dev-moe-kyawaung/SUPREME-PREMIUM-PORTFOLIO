## 🚀 Quick Start

1. Fork/Clone this repository
2. Enable GitHub Pages in Settings > Pages
3. Add your images to `assets/images/`
4. Customize content in `index.html`
5. Deploy instantly!

## 📁 File Structure

```
├── index.html          # Main portfolio
├── style.css           # Supreme animations
├── script.js           # Advanced JS engine
├── assets/             # Images & icons
└── manifest.json       # PWA support
```

## 🎨 Customization

1. **Colors**: Edit CSS custom properties in `:root`
2. **Content**: Update `index.html` sections
3. **Projects**: Add cards in Projects section
4. **Skills**: Modify skill bars data-width
5. **Theme**: Toggle works automatically
